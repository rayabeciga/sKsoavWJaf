# 项目简介

## 核心功能
- 基于YOLOv8的电力设备异常检测
- 支持图片、视频及摄像头三种输入方式进行检测
- 提供UI界面，实时显示目标位置、目标总数和置信度
- 支持批量检测并查看所有结果
- 支持检测结果保存功能
- 可自定义置信度和IOU阈值
- 可选择是否显示标签和原图
- 可切换不同的检测模型

## 适用场景
电力设备异常检测，适用于变电站、输电线路等场景的设备状态监控

## 技术特点
- 使用Python 3.9开发，依赖OpenCV、PyQt5、Ultralytics和Torch1.9等技术栈
- 包含完整的源码、UI界面源文件、数据集图片和运行教程
- 提供500张标注数据集，包含3类电力设备样本
- 已训练好模型，配置环境后可直接使用
